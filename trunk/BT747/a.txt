<tr><th>Option</th><td>Description</td></tr>
<tr><th>------</th><td>-----------</td></tr>
<tr><th>-R</th><td>Recover from disabled log: erase data and reset recording criteria</td></tr>
<tr><th>--UTC <Integer: UTCoffset></th><td>Define UTC offset to apply to output file</td></tr>
<tr><th>-a</th><td>Read all the log memory (overlapped data)</td></tr>
<tr><th>-b <filename.bin></th><td>Do not read device, read a previously saved file.The file type is selected according to the filename extension. Recognized file extensions are .csv, .trl,.nmea, .nme, .nma, .txt, .log, .sr .</td></tr>
<tr><th>--badcolor <HEXCOLOR></th><td>Color to use for 'bad part' in tracks (HEX RGB value), ex 00FFFF</td></tr>
<tr><th>--color <HEXCOLOR></th><td>Color to use for tracks (HEX RGB value, ex 00FF00)</td></tr>
<tr><th>-d <Integer: DEBUG_LEVEL></th><td>Debug level: 0..2</td></tr>
<tr><th>--device <DEVICE></th><td>Make sure the raw bin file is correctly interpreted (DEFAULT, HOLUX).</td></tr>
<tr><th>-f <filename></th><td>Base name for saved files (.bin and other)</td></tr>
<tr><th>-h</th><td>Displays help</td></tr>
<tr><th>--height-to-msl</th><td>Correct WGS84 height (elevation) to MSL (Mean Sea Level)</td></tr>
<tr><th>-l <(on|off)></th><td>Turn logging ON/OFF</td></tr>
<tr><th>-m <(stop|overlap)></th><td>Set STOP/OVERLAP recording method on memory full</td></tr>
<tr><th>-o <log_format></th><td>Enable or disable log fields (FIELD1,- FIELD2,...), available fields: UTC, VALID,LATITUDE,LONGITUDE,HEIGHT, SPEED,HEADING,DSTA,DAGE,PDOP,HDOP, VDOP,NSAT,SID,ELEVATION,AZIMUTH,SNR, RCR,MILLISECOND,DISTANCE,VALID_ONLY</td></tr>
<tr><th>--outtype <OUTPUTTYPE></th><td>Create a gpx file of type NMEA, GPX, GMAP, KML, KMZ, CSV, PLT, TRK.More than one format can be specified when separated with ','</td></tr>
<tr><th>-p <port></th><td>Communication port, default: /dev/ttyUSB0</td></tr>
<tr><th>-r <Integer: time:distance:speed></th><td>Set logging criteria (zero to disable)</td></tr>
<tr><th>-s <Integer: speed></th><td>Serial port speed, default 115200 baud</td></tr>
<tr><th>--splittype <SPLITTYPE></th><td>The way to split the input data: NOSPLIT, DAY or TRACK</td></tr>
<tr><th>-t</th><td>Create a gpx file with tracks</td></tr>
<tr><th>--timesplit <Integer: MINUTES></th><td>Time separation in minutes needed for track segment or track separation.</td></tr>
<tr><th>--trkptinfo</th><td>Add record information for each trackpoint.</td></tr>
<tr><th>--trkptname</th><td>Give each trackpoint a name (based on time)</td></tr>
<tr><th>-v</th><td>Print BT747 version and exit</td></tr>
<tr><th>-w</th><td>Create a gpx file with waypoints</td></tr>
